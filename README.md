# Configuração do Cluster
Acesse o site da [HIVEMQ](https://www.hivemq.com/) , faça a criação da sua conta para continuar no processo de conexão do site com o MQTT, através das etapas a seguir.
- Todas as etapas a seguir foram retiradas da Fonte [Oficial](https://www.hivemq.com/blog/how-to-get-started-with-mqtt/) , caso queria ver e ter mais esclarecimentos.
